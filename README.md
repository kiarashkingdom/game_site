# 🎮 Game Hub

A collection of 4 simple web-based games.
---
## 🎯 Games

- ✊ Rock Paper Scissors
- 🔢 Guess the Number  
- 🎲 Dice Roller
- 🧠 Memory Card Game
---
## 🚀 Run Locally

```bash
pip install -r requirements.txt
python app.py
Then open http://127.0.0.1:5000
```
---

Python + Flask
